web technologies: the technology that used to design and develop the web pages 
1. html->hypertext makup language-structure of web pages
2.css-> cascading style sheet-design/style the web pages
3.js->javascript-logic/functionality of web pages

 html:hypertext markup language-structure of web pages
 :used to provide structure/layout of web pages
 :introduced by tim berners-lee
 Html will save with .html extensions

 Tags: tags are used to  define the structure of the web pages
    : tags are two types
        1.paired tags/container tags
        2.unpaired tags/non container tags/self closing tags

paired tags:it consists of opening tag and closing tag
    <tagname>content</tagname>
    ex: div,p,i,small,heading tags, etc

unpaired tags:it consists of only opening tag 
    <tagname/>
    ex:img,br,hr,etc 

elemets: elemet is a combination of opening tag,closing tag ,and a content
    <tagname>content</tagname>

comment tags: it is used to add comments in the web pages
            : the content of the comment tags will not be displayed in the web pages
            <!--comment content-->

attributes : are used to add additional information to the tags
           : the content of the attributes will not be displayed in the web pages
           ex: src,href,alt,title,style etc
        
block level elemets: the content of block level elements will be displayed in a new line
                    : it occupies the full width of the pages
                    ex: div,p,h,ul,ol etc
        
Inline level elemets: the content of inline level elements will be displayed in a same line
                    : it does not occupies the full width of the pages
                    :it occupies the required space
                    ex: span,a,img,input,button, etc

List: the content of the list will be displayed in a new line
    1.orderd list
    2.unorderd list
    3.data definition/description list
    4.nested list
