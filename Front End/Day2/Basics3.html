Ordered Lists

It is used to display a list in a specicific order & ol tag is used to create an ordered list

The default order is number(1,2,3)

The type attr helps to change the order(type="A" "a" "i" "I" "1")

The reversed attr helps to reverse the order

The start attr helps to start the order from a specific number
